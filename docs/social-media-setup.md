## SECTION 1: ACCOUNT BASICS — Use these on BOTH Facebook and Instagram

### Page Name (Display Name)
```
مشوارو
```

### Username / Handle
Use this exact handle on both platforms:
```
mishwaro.ps
```
- Facebook: `facebook.com/mishwaro.ps`
- Instagram: `@mishwaro.ps`

### Page Category
- **Facebook:** Select "Transportation Service" or "App Page"
- **Instagram:** Select "Transportation"

### Account Type
- **Facebook:** Business Page (NOT personal profile)
- **Instagram:** Business Account (convert from personal in Settings → Account → Switch to Professional)

### Profile Photo (use the SAME image on both)
- **File:** Use the app icon file (1024×1024 PNG)
- **Size for Facebook:** Will display 170×170px on desktop, 128×128px on mobile
- **Size for Instagram:** Will display 110×110px
- Upload the highest quality version (1024×1024) — both platforms compress automatically
- **Source file:** `mishwaro_real_icon_512.png` (or 1024 version)

### Cover Photo / Header
- **Facebook cover photo:** 851×315 px (PNG or JPG)
- **Instagram:** No cover photo, but the bio is the header
- Use the Mishwaro brand colors: forest green `#1a3d2a`, gold `#c9a227`, cream `#faf5e6`
- Recommended cover photo design: Arabic logo + tagline + small Palestinian city skyline silhouette
- If you need to commission this, brief the designer: "Forest green background, gold Arabic text 'مشوارو — شارك الطريق، وفر أكثر', subtle map of West Bank cities."

## SECTION 8: ENGAGEMENT RULES

### Respond to comments
- Within 4 hours during business hours
- Always thank users for positive comments
- For complaints, reply publicly with: "نأسف لتجربتك. راسلنا عبر support@mishwaro.com وسنحلها فوراً." Then move the conversation to DM/email.

### Story posting
- Post 2-3 stories per day after launch week
- Mix: app screenshots, screenshots of bookings (with permission), brand quotes in Arabic, behind the scenes

### Avoid in all posts
- ❌ Politics or sensitive geopolitical topics
- ❌ Comparisons that name competitors directly (don't mention "Careem" or any specific competitor by name in posts — only in comments if asked)
- ❌ Discounts or promo codes (we don't have a promo system yet)
- ❌ Sharing user phone numbers, license plates, or personal info

## SECTION 9: LINKING ACCOUNTS

### Link Instagram to Facebook
1. On Facebook Business Suite, go to Settings → Linked Accounts
2. Connect the Instagram account `@mishwaro.ps`
3. This lets you post to both platforms at once and use Facebook Ads to promote Instagram posts

### Add Facebook Pixel to the website
Send the Facebook Pixel ID to the developer. They'll add it to mishwaro.com so we can run ads later targeting people who visited the website.
